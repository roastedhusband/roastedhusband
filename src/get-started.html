Roasted Husband for your Website

Use this method to get the default Roasted Husband CSS.

  1. Download the {{ site.fontawesome.rh_fnt_prefix }} zip file.
  2. Uncompress the zip file.
  3. Copy the entire {{ site.fontawesome.rh_fnt_prefix }} directory into your website project.
  4. In the <head> of your html, reference the location to your {{ site.fontawesome.rh_fnt_prefix }}.min.css. {% highlight html %}{% endhighlight %}

You can place {{ site.fontawesome.site_title }} icons just about anywhere with the <i> tag, using the CSS Prefix rh and the icon's english name.

Roasted Husband for your Desktop

Follow these steps to use Roasted Husband on the desktop.

  1. Download the {{ site.fontawesome.rh_fnt_prefix }} zip file.
  2. Uncompress the zip file.
  3. Install {{ site.fontawesome.rh_fnt_prefix }}.ttf or {{ site.fontawesome.rh_fnt_prefix }}.otf. You find the font files in the fonts directory of the uncompressed zip file.
  4. Set {{ site.fontawesome.rh_fnt_prefix }} as the font in your application.
  5. Copy and paste the icons (not the unicode) directly from the cheatsheet on this page into your designs.

使用软件编辑Roasted Husband

您可以根据以下的步骤在您的电脑软件里编辑Roasted Husband图标.

  1. 下载 {{ site.fontawesome.rh_fnt_prefix }} zip file.
  2. 解压文件.
  3. 安装 {{ site.fontawesome.rh_fnt_prefix }}.ttf 或者 {{ site.fontawesome.rh_fnt_prefix }}.otf. 您将会直接在fonts文件夹里找到已经解压的文件.
  4. 设置 {{ site.fontawesome.rh_fnt_prefix }} 为您的字体.
  5. 直接在参照表里复制我们的图标，粘贴到您的设计里.
