Reporting Bugs

Found a problem with {{ site.fontawesome.site_title }}? Feel free to submit an issue on the GitHub project. But please keep the following in mind:

  1. Please be nice. {{ site.fontawesome.site_title }} is a happy place.
  2. Please search to see if your bug was already reported.
  3. Before opening any issue, please read the generic issue guidelines, by Nicolas Gallagher.
  4. After doing everything above, feel free to submit an issue.
