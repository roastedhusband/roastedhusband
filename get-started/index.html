  Get Started 使用

Use Roasted Husband 1.0.0 on your website or with your desktop app like InDesign, Photoshop, Word or any other app where you can choose a font.
您可以在任何的网页, 或者图像编辑软件诸如InDesign, Photoshop, Word如使用Roasted Husband 1.0.0.

Roasted Husband for your Website

Use this method to get the default Roasted Husband CSS.

  1. Download the roasted-husband zip file.
  2. Uncompress the zip file.
  3. Copy the entire roasted-husband directory into your website project.
  4. In the <head> of your html, reference the location to your roasted-husband.min.css. 
    <link rel="stylesheet" href="path/to/roasted-husband/css/roasted-husband.min.css">

You can place Roasted Husband icons just about anywhere with the <i> tag, using the CSS Prefix rh and the icon's english name.

<i class="rh rh-bean-curd"></i> Delicious Bean Curd

Delicious Bean Curd


To increase icon sizes relative to their container, use the rh-lg (33% increase), rh-2x, rh-3x, rh-4x, or rh-5x classes.

Roasted Husband for your Desktop

Follow these steps to use Roasted Husband on the desktop.

  1. Download the roasted-husband zip file.
  2. Uncompress the zip file.
  3. Install roasted-husband.ttf or roasted-husband.otf. You find the font files in the fonts directory of the uncompressed zip file.
  4. Set roasted-husband as the font in your application.
  5. Copy and paste the icons (not the unicode) directly from the cheatsheet on this page into your designs.

使用软件编辑Roasted Husband

您可以根据以下的步骤在您的电脑软件里编辑Roasted Husband图标.

  1. 下载 roasted-husband zip file.
  2. 解压文件.
  3. 安装 roasted-husband.ttf 或者 roasted-husband.otf. 您将会直接在fonts文件夹里找到已经解压的文件.
  4. 设置 roasted-husband 为您的字体.
  5. 直接在参照表里复制我们的图标，粘贴到您的设计里.
